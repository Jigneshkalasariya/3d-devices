# 3D Models Directory

Place your GLB model files here:

- `device1.glb`
- `device2.glb`
- `device3.glb`

## Where to Get Models

See the `3D-MODELS-GUIDE.md` in the root directory for sources and instructions.

## Quick Links

- [Poly Pizza](https://poly.pizza/) - Free 3D models
- [Sketchfab](https://sketchfab.com/) - Large collection (filter by downloadable)
- [glTF Viewer](https://gltf-viewer.donmccurdy.com/) - Preview your GLB files

## Notes

- Models should be in GLB format (binary glTF)
- Keep file sizes under 5MB for best performance
- Models will be positioned automatically in the 3D scene
